My projects

    # You can create a new todo by clicking 'New todo' button.
    # 'My Todos' is your default project. You can add more projects by clicking 'Add' button.
    # Once you mark a todo as completed it will be moved to the 'Completed' project.
    # Neither 'My Todos' nor 'Completed' projects can be deleted or renamed.
    # You can delete a project by clicking 'Delete' button.
    # You can edit a project by clicking 'Edit' button.
    # You can sort todos by date created clicking on 'Todo' column header.
    # You can sort todos by due date clicking on 'Due date' column header.
    # You can sort todos by priority clicking on 'Priority' column header.
    # You can sort todos by completion clicking on 'Done' column header.
    # You can filter todos by project clicking on project name.
    # You can open, edit and delete todos by clicking on todo name.
    # You can clear your todo list by clicking 'Clear all' button.
